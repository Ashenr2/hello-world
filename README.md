# Marketing Consultancy Manager

A desktop application built with Electron and React to help manage marketing consultancy work.

## Features

- **Client Management**: Track client information, contacts, and details
- **Campaign Planning & Tracking**: Plan campaigns, set goals, track performance and budgets
- **Content Calendar**: Schedule and manage content across different platforms
- **Reports & Analytics**: Generate insights and view analytics across all your marketing operations

## Installation

1. Install dependencies:
```bash
npm install
```

2. Run in development mode:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
npm start
```

## Technologies Used

- **Electron**: Cross-platform desktop application framework
- **React**: UI library for building the interface
- **Vite**: Fast build tool and development server
- **Local JSON Storage**: Data persistence using Electron's userData directory

## Usage

The application provides an intuitive interface with:
- Dashboard for quick overview of all activities
- Client management for tracking all your clients
- Campaign planning tools with status tracking
- Content calendar for scheduling across platforms
- Comprehensive reporting and analytics

All data is stored locally on your machine in the application's userData directory.
